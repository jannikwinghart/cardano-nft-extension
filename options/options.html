Custom project metadata

How it works

Currently only a few projects are supported by default. You can add projects to your local configuration by uploading a custom metadata file.
To create a custom metadata file you can download the default project metadata below and use it as a template. You can upload your custom metadata file below.

Changes on the custom metadata file will only be made to your local configuration. If you want a project to be added to the default configuration for all users during the next update, contact me on discord (link below).
